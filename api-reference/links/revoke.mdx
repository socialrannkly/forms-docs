---
title: "Revoke Link"
api: "DELETE /external-api/v1/forms/{formId}/links/{linkId}"
description: "Revoke a private form link, immediately preventing it from being used."
---

## Request

### Headers

| Header | Value | Required |
|---|---|---|
| `X-API-Key` | Your API key (`rfk_...`) | Yes |
| `X-Team-Id` | Numeric team ID (required for global API keys) | Conditional |

### Path Parameters

<ParamField path="formId" type="string" required>
  The MongoDB ObjectId of the form (24-character hex string).
</ParamField>

<ParamField path="linkId" type="string" required>
  The unique identifier of the link to revoke.
</ParamField>

## Response

### Success Response

```json 200
{
  "success": true,
  "message": "Link revoked successfully",
  "data": {}
}
```

### Error Responses

<Tabs>
  <Tab title="404 Not Found / Already Revoked">
    ```json
    {
      "success": false,
      "message": "Link not found or already revoked"
    }
    ```
    The link ID doesn't exist, doesn't belong to your team, or was already revoked.
  </Tab>
  <Tab title="401 Unauthorized">
    ```json
    {
      "success": false,
      "message": "Invalid or expired API key."
    }
    ```
  </Tab>
</Tabs>

## What Happens After Revocation

<Steps>
  <Step title="Link is deactivated">
    The link's `isActive` flag is set to `false` in the database.
  </Step>
  <Step title="URL stops working">
    If a customer tries to open the link URL, they will see a "This form is private" error page.
  </Step>
  <Step title="Contact data preserved">
    The associated contact record is **not** deleted. Customer data remains in your contacts.
  </Step>
  <Step title="Existing submissions preserved">
    Any submissions already made via this link are **not** affected.
  </Step>
</Steps>

<Warning>
  Revoking a link is **immediate and irreversible**. The customer will no longer be able to access the form via this link. To restore access, generate a new link.
</Warning>
