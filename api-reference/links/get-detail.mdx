---
title: "Get Link Detail"
api: "GET /external-api/v1/forms/{formId}/links/{linkId}"
description: "Retrieve detailed information about a specific private form link."
---

## Request

### Headers

| Header | Value | Required |
|---|---|---|
| `X-API-Key` | Your API key (`rfk_...`) | Yes |
| `X-Team-Id` | Numeric team ID (required for global API keys) | Conditional |

### Path Parameters

<ParamField path="formId" type="string" required>
  The MongoDB ObjectId of the form (24-character hex string).
</ParamField>

<ParamField path="linkId" type="string" required>
  The unique identifier of the link to retrieve.
</ParamField>

### Example Request

```bash
curl -X GET https://forms.rannkly.com/external-api/v1/forms/665a1b2c3d4e5f6a7b8c9d0e/links/665a1b2c3d4e5f6a7b8c9d12 \
  -H "X-API-Key: rfk_your_api_key"
```

## Response

<ResponseField name="data" type="object">
  <Expandable title="Link Detail Object">
    <ResponseField name="id" type="string">
      Unique identifier of the link.
    </ResponseField>
    <ResponseField name="token" type="string">
      The full access token (starts with `tkn_`).
    </ResponseField>
    <ResponseField name="label" type="string | undefined">
      The internal label assigned to this link.
    </ResponseField>
    <ResponseField name="contactId" type="string | undefined">
      The linked contact's ID.
    </ResponseField>
    <ResponseField name="isActive" type="boolean">
      Whether the link is currently active.
    </ResponseField>
    <ResponseField name="usageCount" type="integer">
      Number of times this link has been used to successfully submit the form. Page views do not count.
    </ResponseField>
    <ResponseField name="maxUsages" type="integer | undefined">
      Maximum allowed submissions. The link becomes inactive when `usageCount` reaches this value.
    </ResponseField>
    <ResponseField name="expiresAt" type="string | undefined">
      ISO 8601 expiration timestamp.
    </ResponseField>
    <ResponseField name="lastUsedAt" type="string | undefined">
      ISO 8601 timestamp of last usage.
    </ResponseField>
    <ResponseField name="createdAt" type="string">
      ISO 8601 creation timestamp.
    </ResponseField>
  </Expandable>
</ResponseField>

### Error Response

```json 404
{
  "success": false,
  "message": "Link not found"
}
```

<Tip>
  Use this endpoint to check the current status of a link — whether it's still active, how many times it's been submitted, and when it was last accessed.
</Tip>
