---
title: "Response Categories"
api: "GET /external-api/v1/feedback/response-categories"
description: "Get a flat list of unique analysis category names found in feedback submissions."
---

This endpoint returns all unique category names extracted from AI feedback analysis, sorted alphabetically. Use it to populate filter dropdowns or discover which categories exist in the data.

## Request

### Headers

| Header | Value | Required |
|---|---|---|
| `X-API-Key` | Your API key (`rfk_...`) | Yes |
| `X-Team-Id` | Numeric team ID (required for global API keys) | Conditional |

### Query Parameters

<ParamField query="startDate" type="string" required>
  Start of the date range. Must be a valid ISO 8601 datetime string.

  **Example:** `2026-01-01T00:00:00.000Z`
</ParamField>

<ParamField query="endDate" type="string" required>
  End of the date range. Must be a valid ISO 8601 datetime string. Must be after `startDate`.

  **Example:** `2026-02-19T23:59:59.999Z`
</ParamField>

<ParamField query="formIds" type="string">
  Comma-separated list of form IDs (MongoDB ObjectIds) to filter by. If omitted, includes all forms accessible to your API key.

  **Example:** `665a1b2c3d4e5f6a7b8c9d0e,665a1b2c3d4e5f6a7b8c9d0f`
</ParamField>

<ParamField query="locationIds" type="string">
  Comma-separated list of location IDs (numeric) to filter by. Only forms assigned to these locations will be included.

  **Example:** `1561,1562`
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  `true` if the request was successful.
</ResponseField>

<ResponseField name="message" type="string">
  `"Response categories retrieved successfully"`
</ResponseField>

<ResponseField name="data" type="string[]">
  An alphabetically sorted array of unique category names (lowercase).
</ResponseField>

### Example Response

```json 200
{
  "success": true,
  "message": "Response categories retrieved successfully",
  "data": [
    "accessibility",
    "accommodation",
    "activities",
    "ambience",
    "amenities",
    "bathroom",
    "cleanliness",
    "food",
    "service"
  ]
}
```

### Empty Response

When no matching AI-analyzed submissions are found:

```json 200
{
  "success": true,
  "message": "Response categories retrieved successfully",
  "data": []
}
```

## Error Responses

### Validation Error (400)

```json 400
{
  "success": false,
  "message": "Validation failed",
  "errors": [
    { "field": "startDate", "message": "startDate is required" }
  ]
}
```

### Missing API Key (401)

```json 401
{
  "success": false,
  "message": "API key is required. Provide via X-API-Key header."
}
```
