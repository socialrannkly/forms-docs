---
title: "Sentiment Overview"
api: "GET /external-api/v1/feedback/sentiment-overview"
description: "Get a breakdown of overall sentiment categories — positive, negative, neutral, and mixed."
---

This endpoint groups AI-analyzed submissions by their overall sentiment label and returns the count for each category. Only submissions that have completed AI analysis with a valid sentiment are included.

Possible sentiment categories:

- **Positive** — respondent expressed a favorable opinion
- **Negative** — respondent expressed an unfavorable opinion
- **Neutral** — respondent expressed neither positive nor negative sentiment
- **Mixed** — respondent expressed both positive and negative sentiment

## Request

### Headers

| Header | Value | Required |
|---|---|---|
| `X-API-Key` | Your API key (`rfk_...`) | Yes |
| `X-Team-Id` | Numeric team ID (required for global API keys) | Conditional |

### Query Parameters

<ParamField query="startDate" type="string" required>
  Start of the date range. Must be a valid ISO 8601 datetime string.

  **Example:** `2026-01-01T00:00:00.000Z`
</ParamField>

<ParamField query="endDate" type="string" required>
  End of the date range. Must be a valid ISO 8601 datetime string. Must be after `startDate`.

  **Example:** `2026-02-19T23:59:59.999Z`
</ParamField>

<ParamField query="formIds" type="string">
  Comma-separated list of form IDs (MongoDB ObjectIds) to filter by. If omitted, includes all forms accessible to your API key.

  **Example:** `665a1b2c3d4e5f6a7b8c9d0e,665a1b2c3d4e5f6a7b8c9d0f`
</ParamField>

<ParamField query="locationIds" type="string">
  Comma-separated list of location IDs (numeric) to filter by. Only forms assigned to these locations will be included.

  **Example:** `1561,1562`
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  `true` if the request was successful.
</ResponseField>

<ResponseField name="message" type="string">
  `"Sentiment overview retrieved successfully"`
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Data Properties">
    <ResponseField name="positive" type="integer">
      Count of submissions with positive sentiment.
    </ResponseField>
    <ResponseField name="negative" type="integer">
      Count of submissions with negative sentiment.
    </ResponseField>
    <ResponseField name="neutral" type="integer">
      Count of submissions with neutral sentiment.
    </ResponseField>
    <ResponseField name="mixed" type="integer">
      Count of submissions with mixed sentiment.
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Response

```json 200
{
  "success": true,
  "message": "Sentiment overview retrieved successfully",
  "data": {
    "positive": 9534,
    "negative": 851,
    "neutral": 146,
    "mixed": 32
  }
}
```

### Empty Response

When no matching AI-analyzed submissions are found:

```json 200
{
  "success": true,
  "message": "Sentiment overview retrieved successfully",
  "data": {
    "positive": 0,
    "negative": 0,
    "neutral": 0,
    "mixed": 0
  }
}
```

## Error Responses

### Validation Error (400)

```json 400
{
  "success": false,
  "message": "Validation failed",
  "errors": [
    { "field": "startDate", "message": "startDate is required" }
  ]
}
```

### Missing API Key (401)

```json 401
{
  "success": false,
  "message": "API key is required. Provide via X-API-Key header."
}
```
