---
title: "NPS Trend"
api: "GET /external-api/v1/feedback/nps-trend"
description: "Get NPS category breakdown (promoter, passive, detractor) over time or by business location."
---

This endpoint groups NPS submissions and returns a count and percentage breakdown of promoter, passive, and detractor categories for each group. Only submissions with a valid NPS category (derived from the NPS question answer) are included.

### Grouping Modes

Use the `groupBy` query parameter to control the output format:

**`groupBy=date`** (default) — Groups by dynamic time interval based on the date range:

| Date Range | Interval | Label Example |
|---|---|---|
| ≤ 1 day | Hour | `14:00` |
| ≤ 1 week | Day of week | `Monday` |
| ≤ 1 month | Date | `10 Jan` |
| ≤ 2 years | Month | `Jan 2026` |
| > 2 years | Year | `2026` |

**`groupBy=business`** — Groups by business location ID, sorted by total responses descending.

## Request

### Headers

| Header | Value | Required |
|---|---|---|
| `X-API-Key` | Your API key (`rfk_...`) | Yes |
| `X-Team-Id` | Numeric team ID (required for global API keys) | Conditional |

### Query Parameters

<ParamField query="startDate" type="string" required>
  Start of the date range. Must be a valid ISO 8601 datetime string.

  **Example:** `2026-01-01T00:00:00.000Z`
</ParamField>

<ParamField query="endDate" type="string" required>
  End of the date range. Must be a valid ISO 8601 datetime string. Must be after `startDate`.

  **Example:** `2026-02-19T23:59:59.999Z`
</ParamField>

<ParamField query="formIds" type="string">
  Comma-separated list of form IDs (MongoDB ObjectIds) to filter by.

  **Example:** `665a1b2c3d4e5f6a7b8c9d0e,665a1b2c3d4e5f6a7b8c9d0f`
</ParamField>

<ParamField query="locationIds" type="string">
  Comma-separated list of location IDs (numeric) to filter by.

  **Example:** `1561,1562`
</ParamField>

<ParamField query="groupBy" type="string">
  Grouping mode. Either `date` (default) or `business`.

  **Default:** `date`
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  `true` if the request was successful.
</ResponseField>

<ResponseField name="message" type="string">
  `"NPS trend retrieved successfully"`
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Data Properties">
    <ResponseField name="response" type="array">
      Array of grouped NPS trend items.
      <Expandable title="Item Properties">
        <ResponseField name="total_responses" type="integer">
          Total NPS submissions in this group.
        </ResponseField>
        <ResponseField name="categories" type="object">
          Breakdown per NPS category.
          <Expandable title="Category Properties">
            <ResponseField name="promoter" type="object">
              `{ "count": integer, "percent": number }`
            </ResponseField>
            <ResponseField name="passive" type="object">
              `{ "count": integer, "percent": number }`
            </ResponseField>
            <ResponseField name="detractor" type="object">
              `{ "count": integer, "percent": number }`
            </ResponseField>
          </Expandable>
        </ResponseField>
        <ResponseField name="date_time" type="string">
          Date label (only present when `groupBy=date`).
        </ResponseField>
        <ResponseField name="business_id" type="integer">
          Location ID (only present when `groupBy=business`).
        </ResponseField>
        <ResponseField name="business_nickname" type="string | null">
          Business name (only present when `groupBy=business`; currently `null`).
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="categories" type="string[]">
      Alphabetically sorted list of NPS categories present in the result.
    </ResponseField>
  </Expandable>
</ResponseField>
