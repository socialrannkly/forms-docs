---
title: "Department List"
api: "GET /external-api/v1/feedback/departments"
description: "Retrieve a list of available departments linked to NPS questions in feedback forms."
---

This endpoint returns unique departments that are linked to NPS-type questions across feedback forms. Useful for populating department filter dropdowns in dashboards.

### How It Works

1. Finds all feedback forms matching the filters (team, locations).
2. Extracts `departmentId` from NPS-type questions in those forms.
3. Looks up department names from the Department collection.
4. Returns unique departments sorted alphabetically by name.

## Request

### Headers

| Header | Value | Required |
|---|---|---|
| `X-API-Key` | Your API key (`rfk_...`) | Yes |
| `X-Team-Id` | Numeric team ID (required for global API keys) | Conditional |

### Query Parameters

<ParamField query="locationIds" type="string">
  Comma-separated list of location IDs (numeric) to filter forms by.

  **Example:** `1561,1562`
</ParamField>

### Example Request

<Tabs>
  <Tab title="cURL">
    ```bash
    curl -X GET "https://forms-api.rannkly.com/external-api/v1/feedback/departments?locationIds=1561" \
      -H "X-API-Key: rfk_your_api_key"
    ```
  </Tab>
  <Tab title="JavaScript">
    ```javascript
    const response = await fetch(
      "https://forms-api.rannkly.com/external-api/v1/feedback/departments?" +
      new URLSearchParams({ locationIds: "1561" }),
      { headers: { "X-API-Key": process.env.RANNKLY_API_KEY } }
    );
    const { data } = await response.json();
    ```
  </Tab>
  <Tab title="Python">
    ```python
    import requests

    response = requests.get(
        "https://forms-api.rannkly.com/external-api/v1/feedback/departments",
        headers={"X-API-Key": "rfk_your_api_key"},
        params={"locationIds": "1561"}
    )
    data = response.json()
    ```
  </Tab>
</Tabs>

## Response

<ResponseField name="success" type="boolean">
  `true` if the request was successful.
</ResponseField>

<ResponseField name="message" type="string">
  `"Department list retrieved successfully"`
</ResponseField>

<ResponseField name="data" type="array">
  Array of department objects sorted alphabetically by name.
  <Expandable title="Department Properties">
    <ResponseField name="uid" type="string">
      Department ID (MongoDB ObjectId).
    </ResponseField>
    <ResponseField name="name" type="string">
      Department name.
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Response

```json 200
{
  "success": true,
  "message": "Department list retrieved successfully",
  "data": [
    {
      "uid": "65a1b2c3d4e5f6a7b8c9d0e1",
      "name": "food and beverage service"
    },
    {
      "uid": "65a1b2c3d4e5f6a7b8c9d0e2",
      "name": "housekeeping"
    }
  ]
}
```

### Empty Response

When no NPS questions with departments are found:

```json 200
{
  "success": true,
  "message": "Department list retrieved successfully",
  "data": []
}
```

## Error Responses

### Missing API Key (401)

```json 401
{
  "success": false,
  "message": "API key is required. Provide via X-API-Key header."
}
```
