---
title: "Rating Overview"
api: "GET /external-api/v1/feedback/rating-overview"
description: "Get a summary count for each star rating (1-5) and the overall average rating."
---

This endpoint provides a high-level summary of ratings across all matching form submissions. It returns the count of responses per star rating (1 through 5) and the overall average rating.

<Note>
  Ratings are always normalized to a **1-5 scale** regardless of the original question type. NPS (0-10) and opinion scale (1-10) values are mapped proportionally to the 1-5 range before averaging.
</Note>

## Request

### Headers

| Header | Value | Required |
|---|---|---|
| `X-API-Key` | Your API key (`rfk_...`) | Yes |
| `X-Team-Id` | Numeric team ID (required for global API keys) | Conditional |

### Query Parameters

<ParamField query="startDate" type="string" required>
  Start of the date range. Must be a valid ISO 8601 datetime string.

  **Example:** `2026-01-01T00:00:00.000Z`
</ParamField>

<ParamField query="endDate" type="string" required>
  End of the date range. Must be a valid ISO 8601 datetime string. Must be after `startDate`.

  **Example:** `2026-02-19T23:59:59.999Z`
</ParamField>

<ParamField query="formIds" type="string">
  Comma-separated list of form IDs (MongoDB ObjectIds) to filter by. If omitted, includes all forms accessible to your API key.

  **Example:** `665a1b2c3d4e5f6a7b8c9d0e,665a1b2c3d4e5f6a7b8c9d0f`
</ParamField>

<ParamField query="locationIds" type="string">
  Comma-separated list of location IDs (numeric) to filter by. Only forms assigned to these locations will be included.

  **Example:** `1561,1562`
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  `true` if the request was successful.
</ResponseField>

<ResponseField name="message" type="string">
  `"Rating overview retrieved successfully"`
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Data Properties">
    <ResponseField name="response" type="object">
      Rating distribution keyed by star value (`"1"` through `"5"`). Each value is the count of responses with that rating. Only ratings with responses are included.

      **Example:** `{ "5": 330, "4": 22, "3": 3, "2": 2, "1": 2 }`
    </ResponseField>
    <ResponseField name="overall_average_rating" type="number">
      The overall average rating across all matching responses, rounded to 1 decimal place. Range: 1.0 to 5.0.
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Response

```json 200
{
  "success": true,
  "message": "Rating overview retrieved successfully",
  "data": {
    "response": {
      "5": 330,
      "4": 22,
      "3": 3,
      "2": 2,
      "1": 2
    },
    "overall_average_rating": 4.9
  }
}
```

### Empty Response

When no matching submissions are found:

```json 200
{
  "success": true,
  "message": "Rating overview retrieved successfully",
  "data": {
    "response": {},
    "overall_average_rating": 0
  }
}
```

## Error Responses

### Validation Error (400)

```json 400
{
  "success": false,
  "message": "Validation failed",
  "errors": [
    { "field": "startDate", "message": "startDate must be before endDate" }
  ]
}
```

### Missing API Key (401)

```json 401
{
  "success": false,
  "message": "API key is required. Provide via X-API-Key header."
}
```

### Global Key Restricted (403)

```json 403
{
  "success": false,
  "message": "This route is not accessible with the global API key."
}
```
